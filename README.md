# GitLabTerminator
Understand GitLab projects quickly.
Find suitable beginner-friendly learning projects.
Get AI-suggested issues and fix steps.
## Needed
GitLab repository URL(https://gitlab.com)<br>
GitLab token([Create GitLab Personal Access Token](https://docs.gitlab.com/user/profile/personal_access_tokens/))<br>
Gemini API key([Create Gemini API Key](https://aistudio.google.com/app/apikey))<br>
